# Détective des Différences

## Description
Un jeu interactif où le joueur doit trouver les différences entre deux images similaires. Le jeu propose trois niveaux de difficulté avec un système de score et de chronomètre.

## Technologies utilisées 
- HTML5
- CSS3 (avec animations et design responsive)
- JavaScript vanilla

## Fonctionnalités principales
- Trois niveaux de difficulté (Facile, Moyen, Difficile)
- Système de comptage des différences trouvées
- Chronomètre
- Limite d'essais par niveau
- Interface responsive et animations

## Nouveautés explorées
- Gestion d'état complexe avec JavaScript
- Création dynamique d'éléments DOM
- Système de coordonnées pour les zones cliquables
- Animations CSS avancées

## Difficultés rencontrées
1. Calibration précise des zones de différence
2. Gestion des événements de clic sur les images
3. Synchronisation du timer avec l'état du jeu

## Solutions apportées
1. Utilisation de pourcentages pour les positions des différences
2. Implémentation d'un système de détection de clic avec coordonnées
3. Gestion propre du timer avec clearInterval
